# AI Study Manager  
### 파이썬 SW 활용 경진대회 출품작

본 프로젝트는 **광운대학교 제1회 「파이썬 SW 활용 경진대회」 제출을 목적으로 개발한 웹 기반 학습 관리 시스템**입니다.
Python을 활용하여 **학습 타이머, 데이터 분석, 시각화, 생성형 AI**를 하나의 서비스로 통합하는 것을 목표로 하였습니다.

---

## 프로젝트 개요
**AI Study Manager**는 뽀모도로 학습 타이머를 기반으로  
실제 학습 행동 데이터를 자동으로 기록·분석하고,  
그 결과를 **Gemini API 기반 학습 코치 리포트**로 제공하는 웹 애플리케이션입니다.

특히, 실제 열람실 환경을 반영한 **좌석 이용 시간 제한 로직**을 구현하여  
현실적인 학습 흐름 관리가 가능하도록 설계하였습니다.


## 주요 기능
- 집중/휴식 타이머 (Pomodoro)
- 열람실 좌석 시간 관리 (23:00 종료 규칙 반영)
- 학습 기록 자동 저장
- 집중 패턴 시각화
- Gemini AI 학습 리포트 생성
